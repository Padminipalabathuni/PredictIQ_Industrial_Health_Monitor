# PredictIQ_Industrial_Health_Monitor
Ai -powered IOT sensor anomaly detection system with health scoring, failure prediction and automated PDF reports
# What it does
- Detects anomalies in industrial sensor data
- Predicts machine failure before it happens
- Generates automated PDF diagnostic reports
- Real-time health scoring (0-100)
- Email alert simulation
- Severity classification

# Technologies
Python | Scikit-learn | Isolation Forest | Pandas | Matplotlib | ReportLab
